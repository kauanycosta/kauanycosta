## 📌 Projeto em destaque

### 🏆 [Tabela do Brasileirão 2025](https://github.com/kauanycosta/tabela-brasileirao)

Tabela interativa do Campeonato Brasileiro de Futebol 2025, desenvolvida com **PHP, HTML, CSS e JavaScript**.

O projeto consome dados de uma **API via PHP** para exibir clubes atualizados, resultados recentes e informações dos próximos jogos, com layout responsivo para desktop e dispositivos móveis.

🔗 **Projeto online:**  
👉 http://tabelabrasileirao.ct.ws

**Funcionalidades:**
- Consumo de API para exibição de dados reais  
- Visualização da tabela com posição, time, pontos e desempenho  
- Cards com últimos resultados e próximo jogo  
- Layout responsivo (mobile e desktop)  
- Interações com hover (desktop) e clique (mobile)
